# v1.7 — locality NEG 选法消融:sibling vs generic-unrelated vs cross-group(measurement-only)

> 依据: v1.6 结论(`docs/v1.6-contrast-out-locality-fix.md`)+ 本轮用户指令
> 类型: **measurement-only** · 只换 NEG 定义**一个**变量,质疑 metric 本身
> 目标: 判断当前 S3 locality 失效(sibling-NEG NEGxfire fact 11/20)有多少是 **NEG 选法**(sibling 同构)放大的 artifact,多少是系统**真实污染【无关知识】**的缺陷。

## 背景锚点(别再质疑)
- v1.5/v1.6:S3 chat key 整锥被 scaffold + flat-mean 池化抬到 ~0.88;contrast-out 是 false-fix;facts 在此 key 下彼此**不线性可分**(A0002 sibling-NEG 0.898 > 自己 PARA 0.886)。见 [[horen-key-cone-collapse]]。
- 当前 locality NEG = **同组 sibling 的 DIRECT**(F1 的 NEG = 问 F2)。5 条 fact 的 src 高度同构(都 "What is the [X] of JQ's [Y]"),sibling 之间本就极像 → **可能 NEG 选得太像 DIRECT**,而非真失效。
- ★ ZsRE 的 locality NEG 测的是"编辑会不会污染**【无关邻域知识】**",**不是**"两条相似编辑会不会互串"。两者不是一回事。
- 已有线索(待本轮正式确认):generic-NEG 地板 ~0.78(不误触发)vs sibling-NEG 地板 ~0.88(误触发),阈值 0.85 正卡在中间。
- ⚠️ 开放问题(不预判):scaffold 把**整锥**抬高,而 generic Q **也**过同一个 chat scaffold → NEG-B 有可能**也**高。本轮**测**,不假设。

## 实验(复用 S3 harness:同 10 样本 / 同 seed / 同 `keying.score` / 阈值 0.85,全部不变;**只换 NEG 定义**)
- **NEG-A(现状)**:同组 sibling 的 DIRECT(每 edit 4 个 sibling,5×4 = **20**)。
- **NEG-B(ZsRE 式)**:与所有编辑事实**无关**的 generic 世界知识问题(capital of France / days in a week / ... 模型本就会、与任何编辑事实/信念无关),8 条 × 5 edits = **40**(同一 generic 池对 fact 与 belief 两组通用)。
- **NEG-C(可选,中等相似)**:跨组——fact 的 NEG 用 belief 的 src,belief 的 NEG 用 fact 的 src(5×5 = **25**)。
- **DIRECT / PARA / PARAfire 与 NEG 定义无关**(edit-vs-own-src/para),三条件**恒定**;只 **NEGfloor / NEGxfire** 随 NEG 变。**NEGfloor(最坏相似)= 跨条件最干净的可比量**(与分母无关)。
- generic NEG-B 是**外部通用世界知识探针**(本任务明确要求并给示例),非编辑样本、不写入 `samples.json`,spike 内逐条打印披露。避开食物(belief A0143 是 cheese)与奇幻主题,确保与两组都无关。

## 结果(开发后)

### 落地 & 校验
- spike: `spikes/spike_v17_neg_ablation.py`(measurement-only,独立 model 副本;只换 NEG 定义;**未碰** server/codebook/memory/editing/samples.json)。
- 生产路径零漂移:S3 key == `compute_key(templated=True)`(drift-guard `allclose atol 1e-3` 过);打分 `keying.score`(生产 `_query` gate,阈值 0.85 不变)。
- **完整性闸通过**:NEG-A 精确复现 v1.5/v1.6(fact 11/20、belief 7/20、PARAfire 5/5)→ NEG 定义是**唯一**变量。

### 对照表(阈值 0.85;DIRECT/PARA/PARAfire 与 NEG 无关 = 三条件恒定)
| NEG 条件 | group | DIRECT | PARA | **NEGfloor** | NEGmean | PARAfire | **NEGxfire** |
|---|---|---|---|---|---|---|---|
| **A sibling**(现状) | fact | 0.915 | 0.902 | **0.898** | 0.842 | 5/5 | **11/20 (55%)** |
|  | belief | 0.925 | 0.932 | 0.871 | 0.827 | 5/5 | **7/20 (35%)** |
| **B generic-unrelated** | fact | 0.915 | 0.902 | 0.826 | 0.760 | 5/5 | **0/40** |
|  | belief | 0.925 | 0.932 | 0.837 | 0.778 | 5/5 | **0/40** |
| **C cross-group** | fact | 0.915 | 0.902 | 0.850 | 0.804 | 5/5 | **0/25** |
|  | belief | 0.925 | 0.932 | 0.862 | 0.824 | 5/5 | **3/25 (12%)** |

NEGfloor 随 NEG↔DIRECT 相似度**单调**:sibling > cross > generic(fact 0.898>0.850>0.826;belief 0.871>0.862>0.837)。**0.85 阈值正好卡在 sibling(上)与 generic(下)之间** → NEGxfire 完全是"NEG 选多像 DIRECT"的函数。

### 判读(4 问)
1. **NEG-B(generic 无关)= 0/40(fact、belief 都 0)。** generic floor fact 0.826 / belief 0.837,均 < 0.85。比 NEG-A(fact 11/20 55% / belief 7/20 35%)从 **55%/35% → 0%**;floor 从 0.898/0.871 掉到 0.826/0.837,跨到阈值下。
2. **NEG-B 清零 + NEG-A 高 ⇒ locality 失效主要是 sibling 同构放大的 ARTIFACT;系统对【无关世界知识】没污染。** 产品没那么坏,是用例严苛。
   - ⚠️ **但边际很薄**:generic floor 0.82–0.84,只比阈值低 **~0.01–0.03** —— 整锥被 scaffold 抬起(v1.5),unrelated 是"**刚好**不触发"而非"安全地不触发"。换语言 / 微调阈值(ZH 锥下滑、EN 锥上抬,见 [[horen-key-cone-collapse]])就可能翻车。
3. **fact vs belief**:**sibling 级 fact 更糟**(11 vs 7,fact src 更同构、共享 JQ);但 **generic/cross 级 belief 锥略高**(generic floor 0.837 > 0.826;cross belief 3/25 vs fact 0/25)。即 fact 在"近邻"更易撞,belief 整体锥位略高。
4. **一句话**:触目的 **11/20 主要是 NEG 选法 artifact**(near-duplicate 同主语 sibling);系统**不**乱触发无关知识(NEG-B 0/40);真实缺陷收窄为**"同主语 sibling 互串"**这个更窄的问题,且它坐在一个被抬薄了的锥上(generic 仅差阈值 0.01–0.03)。

### 对前序结论的修正(重要)
- v1.5/v1.6 用 **sibling-NEG** 量的"locality 塌缩",**夸大**了严重性。换成 ZsRE 本义的 **unrelated-NEG**,系统**通过**(0/40)。
- 真问题收窄为:**同一主语(JQ)下多条相似 personal fact 之间的 deferral 互串** —— 这是本产品的真实(但有界)风险:编辑 5 条 JQ 的事实,问其一可能触发其二;**不是**"污染全部世界知识"。
- 修复优先级因此可重估:只关心 unrelated-locality,则当前阈值/键**已够**;要消 sibling 互串,才需 v1.6 末尾那类更强 key 表示。
