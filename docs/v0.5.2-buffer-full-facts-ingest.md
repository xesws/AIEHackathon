# v0.5.2 — ingest(chat) 入口:全量 edit-route 进 buffer + 多事实重验

> 阶段:serving 入口(解锁后第一步)/ 串行 / GPU 常驻。开发前写计划,开发后补结果(遵 CLAUDE.md /docs 规则)。
> 前置已绿:v0.4 hero smoke 跑出 `extract path: real-extract` + must-pass PASS → 真 LLM 上游已证明能用,serving 解锁。
> 本步是"工程搬运":把 spike 里那段 extract→buffer 粘合搬成生产入口,**并焊进两条纪律**(下方)。

## 焊进指令的两条纪律(用户明确要求)
1. **`ingest(chat)` 必须 buffer 全部 edit-route items(for-loop),不是 `[0]`。** v0.4 smoke 只取 `edit_items[0]`,
   把一句话里的第二个事实(过敏)丢了 —— 入口不能再丢。
2. **搬完立刻重验"多事实"场景**:一句话含**两个**事实("我是 JQ,对镍扣过敏")→ ingest → run_pass →
   **专问【过敏】**(不是名字)→ 真答出 `nickel buckles`,而**不是**复述名字的假阳性。这一验通了才算"多事实 ingest"真成立。

## 实现(最小,hero-first)
- **`serving/ingest.py`** 新增 `ingest(chat) -> dict`:`items = extract.extract(chat)`;**for-loop 全量**路由 ——
  `route=="edit"` → `buffer.append(it)`(全部,不取 [0]);`route=="rag"` → `rag_store.add(it)`。返回计数
  (`n_extracted / n_edit_buffered / n_rag_indexed / edit_ids`)。**不**在此 consolidate(那是 triggers.manual 的事)。
- **`serving/triggers.py`** 落地 `manual()` → `consolidate.run_pass("manual")`("Consolidate Now")。
- 不碰 0.85 阈值 / key 空间 / query-span / editing / keying / generate / memory 任何模块的逻辑;不动 FastAPI app.py(下一步)。

## 验证(`spikes/spike_v052_multifact_ingest.py`,真 LLM + 真权重)
1. `load_base()` 一次;`store.reset()`+`rag_store.reset()`;`set_model_provider`。
2. `chat = [{"role":"user","content":"Hey, I'm JQ. Just so you know, I have a severe contact allergy to nickel buckles."}]`。
3. `res = ingest(chat)` → **打印抽出的每条 fact(stem/target/route)**;断言 buffer 里 **≥2 条 edit-route**,且其中
   **有一条过敏 fact**(`provenance['edit'].target` 含 "nickel"/"buckle")、**有一条名字 fact**(target=="JQ")。
   ← 这一步直接证"全量 edit-route 入 buffer,没丢第二个事实"。
4. `n = triggers.manual()`(= run_pass);断言 `n == 入 buffer 的 edit 条数`,buffer 抽干。
5. **专问过敏**:用过敏 fact 的 stem 走 chat(RAG off)→ 答案含 `nickel`/`buckle`(过敏 target),
   **且不是只复述 "JQ"** → 证学到的是过敏、可检索、非假阳性。(附:名字 stem → 答 "JQ"。)
6. 一条无关 query → 不误触发。

## 绿灯判据
- ingest 把**两条** edit-route(名字 + 过敏)都进了 buffer(不止 [0])。
- run_pass 全写入(`n==2`),叠进同一 codebook,buffer 抽干。
- **问过敏 → 真答 nickel buckles**(非名字假阳性);问名字 → 答 JQ;无关 query 不误触发。
- ⚠ 若真 extract **没把过敏抽成 edit-fact**(漏抽/路由去 rag/合并)→ 断言失败 → **如实报告**该发现,不蒙绿。
