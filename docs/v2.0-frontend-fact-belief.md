## 0. 背景

v1.9 后:buffer 只装 **belief**(route==edit);rag_store 装 **fact + other**(都带 `type` 标签);
consolidated = 内化进权重的 belief。§0.3 的灵魂:**fact 在 prompt 里看得见(检索)、belief 在权重里看不见(但答对)**。

## 1. 三处交互改动

1. **buffer/Pending 区 → 重标为 belief**
   - 现文案"新事实 · 建议写入权重" → "信念/观点 · 内化进权重"。buffer 现在只有 belief。
2. **Reference/RAG store → 按 `type` 拆分展示 fact vs other**
   - `refresh()` 把 `m.type` 带到 refs;`MemorySurface` 把 fact(你的个人事实)与 other(参考资料)分组/分标签。
   - LabPanel 的 "reference · RAG store" 同样区分。
3. **★ proof 对比可视化(demo 主角)**
   - 答案来自 **RAG**(retrieved 非空)→ "retrieved · 在 prompt 里"(看得见);
   - 答案来自 **权重**(attribution.hit,rag_off)→ "from weights · 不在 prompt 里"(看不见但答对)。
   - 强化现有 TokenAttribution + retrieved badge,把"fact 显式 / belief 隐式"的对照讲清楚。
