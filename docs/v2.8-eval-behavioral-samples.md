# v2.8 — eval 行为评测数据 (demo_samples 格式, 由 A 池转换)

> 一句话: 把 `eval/samples.json` 的 A 知识池**确定性地**转换成 `demo/demo_samples.json` 的
> 行为格式 (`turns[]` teach→probe + `free_scenario_e2e`), 落盘成 `eval/behavioral_samples.json`,
> 让真实 `/chat` 流水线有一份可扩展到数百条的行为评测数据 (现在 demo 只有 8 条手挑 turn)。

## 背景

仓库里有两套"样本"世界:

- **`eval/` 评测集** — 离线定量基准 (A/B/C 三族, 500 条), 比较 editing-vs-RAG,
  `eval/dataset.py` 加载, 进程内直接跑本地 Llama (不经过 `/chat`)。
- **`demo/demo_samples.json`** — 端到端行为格式: `turns[]` (teach→probe + `expect_type / expect_route /`
  `expect_source / expect_words / ...`) + `free_scenario_e2e`。由 `spikes/spike_v22_dress_rehearsal.py`
  打到真实 `/chat` 上跑 (当前 spike 把 `expect_*` 当声明式文档, 按 gate 硬编码等价判断)。

A 池本就是"关于 JQ 的全部知识"的共享原子 (SCHEMA §0.2), type X/Y 又天然对应 demo 的 belief/fact 路由
(`memory/router.py`: belief→edit, fact/other→rag)。故把 A 池映射成 demo 行为格式即得到行为评测数据。

本轮范围 (用户澄清): **只产出数据文件**, 不写 loader / runner / scorer。

## 转换规则 (确定性, source = 370 条 A)

| A 源 | demo turn | route / source | probe |
|---|---|---|---|
| **type X** (世界观信念, subject=null) — 222 条 | `kind: "belief"`, `expect_type: "belief"` | `expect_route: "edit"` / `weights` | `rag_off: true`, `expect_retrieved_empty: true` |
| **type Y** (JQ 个人事实, subject="JQ") — 148 条 | `kind: "fact"`, `expect_type: "fact"` | `expect_route: "rag"` / `rag` | `rag_off: false`, `expect_retrieved_nonempty: true` |

- **belief.teach** = `edit_prompt.replace("___", target_new)` 填空成句, 前置轮换信念框
  (`Honestly,` / `I'm convinced` / `In my opinion,` / `I really think` / `I believe`), 并把填空句首字母
  小写 (A 池 X 的 edit_prompt 首词皆为 the/to/for/gerund 等普通词, 无专有名词首词, 小写安全)。
- **fact.teach** = `rag_doc` (已是干净的 JQ 归属事实句)。
- **probe.ask** = `queries[0].q`; **probe.expect_words** = `[target_new.lower()]` (子串匹配, 同 demo)。
- 每 turn 附溯源字段 `source_id / key / type / prior_hardness / category` (spike loader 容忍未知字段)。

> `expect_type` / `expect_route` 是 eval 设计意图下的 **gate/gold** (X 定义为 edit 目标, Y 为 rag 目标),
> **不是**喂模型的提示; type 由真 classifier 运行时决定。真实 classifier 是否同意正是这份数据要**测**的。

## 手写补充 (A/B/C 无, demo 格式需要)

- **other × 2** (日程/参考类): `expect_type:"other"`, `expect_route:"rag"`, 仿 demo T3, English-only。
- **transient × 2** (闲聊噪声): 仿 demo T4, 仅 `expect_extracted:0` + `expect_dropped:true`。

## free_scenario_e2e (镜像 demo, 由 3 条 X-belief 派生)

- 选 3 条不同 category 的 real-world X 信念: `A0143` (food_taste / muenster) +
  `A0146` (music_taste / early eighties) + `A0147` (movie_taste / Clash of the Titans) → 复古观影夜场景。
- `setup`: `teach_turn_ids = [A0143-belief, A0146-belief, A0147-belief]`,
  `consolidate {background:true, expect_n_written:3}`, `expect_rag_indexed:0`。
- 2 条组合探针 (各融合其中 2 条信念), `rag_off:true`, `expect_retrieved_empty:true`,
  `expect_source:"scenario_memory_private_lane"`, `expect_selected_contains/excludes` 用 target 子串
  (对齐 `serving/scenario_memory.py` 返回的 `scenario_memories.selected[].text`)。
