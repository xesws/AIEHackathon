## 目标
v0.8 的 serving 三端点已验过(HTTP hero PASS)。前端 `engram.jsx` 仍是 mock-only(0 网络)。本步把

## 关键事实

- 后端 `http://localhost:8077`,CORS `*`。契约:
  - `POST /chat {message, rag_off}` → `{reply, buffer_count, learned}`(learned = edit_id 列表)
  - `POST /consolidate {}` → `{n_written, buffer_count}`

## 映射(接 serving 的三条)
- `send()` → `POST /chat {message, rag_off: !ragOn}`:append user → await → append assistant
  `{text: reply, captured: learned.length?'记下 N 条':undefined}` → `refresh()`。**ragOn→rag_off 在此接(hero proof 开关)。**
- `consolidate()`(LabPanel "consolidate now")+ `onBurnAll`(MemorySurface "全部写入")→ `POST /consolidate {}` → `refresh()`(同一 async)。
- `refresh()` = `GET /memories` → `weights ← consolidated.map({id,text})`、`buffer ← buffer.map({id,text,status:"new"})`。mount + 每次 chat/consolidate 后。
- loading 态:`booting`(mount 轮询 /memories 直到模型就绪,显示"模型加载中…",一次性 readiness gate)、`sending`(/chat in-flight)、`consolidating`(/consolidate in-flight)→ disable 控件 + 文案。

### 部署说明(RunPod:单端口同源,零配置)
浏览器不在 pod 上 → 硬编 `localhost:8077` 不可达;RunPod 每端口是独立 `-<port>.proxy.runpod.net` 子域,
跨端口还会 https→http mixed-content 被拦。**最终方案:单端口同源**,用户侧零配置:
- **后端 `serving/app.py` 用 `StaticFiles` 把 `frontend/` 挂在同一个 app**(API 路由在前、`app.mount("/")` 在后,
  `html=True` 在 `/` 出 index.html)→ 页面与 `/chat /consolidate /memories` **同源**:无 CORS、无 mixed-content、无需填 URL。
- **前端 `API` 默认 `""`(同源相对路径)**;`window.__ENGRAM_API__` 仅用于分离式 dev。
- **跑法**:`uvicorn serving.app:app --host 0.0.0.0 --port 5173`(复用已暴露的 5173 一个端口,bind `0.0.0.0`)。
  用户**重开原来那个 `…-5173.proxy.runpod.net` 链接**即可,RunPod 不用改任何设置。
- 验证(本机):`GET /`→index.html 200、`GET /memories`→JSON、`POST /chat`→reply/buffer_count/learned 正常。

### 无 CDN 自托管(修白屏:浏览器够不着 esm.sh/unpkg/cdn.tailwind)
白屏根因:`index.html` 原本靠浏览器内 Babel + esm.sh import map + Tailwind Play CDN —— 受限网络(够不着这些 CDN)→ 模块加载失败 → 白屏。已改为**全部本地化、零外链**:
- **`frontend/app.bundle.js`**(esbuild 把 `src/engram.jsx` + React + react-dom + lucide 图标打成一个自包含 IIFE,无 in-browser Babel、无 esm.sh)。
- **`frontend/vendor/tailwind.js`**(Tailwind Play CDN 本地副本)。
- `index.html` 只 `<script src="./vendor/tailwind.js">` + `<script src="./app.bundle.js">`,无任何外链;由 FastAPI StaticFiles 从 pod 提供。
- ⚠ **改了 `src/engram.jsx` 必须重建 bundle**:
  `esbuild src/engram.jsx --bundle --format=iife --minify --define:process.env.NODE_ENV='"production"' --outfile=app.bundle.js`
  (本机已下载 esbuild 二进制到 `/tmp/package/bin/esbuild`;npm 包用 `registry.npmjs.org` tarball 临时装到 `node_modules` 后即删)。
- 验证:`GET /app.bundle.js` 200、`GET /vendor/tailwind.js` 200、`index.html` 内无 `https://` 外链;bundle 含 entry `getElementById("root")` + `createRoot` + `rag_off//memories` 等。
