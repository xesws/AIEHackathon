# v2.7 — "Her" 风思考动画(呼吸光球 + 涟漪)

> 本轮目标(一句话):把 chat 里「Engram 正在想…」那条**静态** `<Mark/> + 文字`(`engram.jsx:545`)
> 换成一个《Her》风格的**有机呼吸光球 + 双涟漪**的 thinking 动画 —— jade 暖色调、在品牌内,纯前端、不碰后端。

---

## 一、计划节

### 1. 现状
`ChatSurface` 的 `sending` 指示器(`engram.jsx:545-549`)只是:
```jsx
{sending && (<div className="self-start flex items-center gap-1.5" ...><Mark size={13} /> Engram 正在想…</div>)}
```
一个静态 trace 字形 + 文字,没有"在思考"的生命感。

### 2. 设计(选定:呼吸光球 + 涟漪,Her 原味)
新增一个自包含组件 `ThinkingOrb`,放回 `sending` 指示器位置。视觉元素(全用 `C` 调色板:`jade #0E8C5A` /
`jadeInk #0A6B45` / `trace #3DDC84`,暖纸背景):
- **光球**:`radial-gradient(circle at 35% 30%, trace → jade → jadeInk)` 做出被左上光照的球体高光;
  `@keyframes breathe` 缓慢呼吸(`scale 1 ⇄ 1.12` + jade 辉光 `box-shadow` 浓淡),`2.4s ease-in-out infinite`。
- **内部光晕自转**:球内一层 `conic-gradient` 浅 jade sheen,`@keyframes spin` `5.5s linear` 缓转 → "活着的"流光。
- **双涟漪**:球后两个 jade 细边圆环,`@keyframes ripple`(`scale 0.55→2.0` + `opacity 0.5→0`),`2.8s ease-out infinite`,
  第二道 `animation-delay 1.4s` **错峰**扩散淡出。
- **文字**:`Engram 正在想…` 配 `@keyframes textPulse`(`opacity 0.5 ⇄ 1`)与呼吸同步,催眠感。
- **节奏**:呼吸 2.4s / 涟漪 2.8s 故意**不整除** → 不机械锁步,更像有机体。

`@keyframes` 经组件内渲染的 `<style>{THINKING_CSS}</style>` 注入(自包含,仅 thinking 时挂载;无需改 `build.py`)。

### 3. 护栏 DO NOT
- 只改 `frontend/src/engram.jsx`(加 `ThinkingOrb` + `THINKING_CSS`,替换 `sending` 指示器)+ `build.py` rebuild 产物
  (`app.bundle.js` / `index.html`)+ 本文档。**不碰**任何后端 / `serving` / `memory` / 测试逻辑 / `demo_samples*` / `.env`。
- 改完**必须** `python frontend/build.py` 重新内联(`index.html` 与 source 同步,`test_frontend_smoke` 会查)。
- 纯前端、**不需要 GPU**;只 `git add` 实际改的文件;绝不 `-A`;不 push。

### 4. 验收
- `pytest tests/test_frontend_smoke.py -q` 全绿(组件/wiring 不破 + `index.html` 与 source 同步)。
- 视觉效果(球是否够"Her"、呼吸/涟漪节奏是否顺眼)= **浏览器肉眼**,headless 看不了 → 交用户看一帧,按需调
  速度/大小/颜色。

## 二、结果节

> **结论:`ThinkingOrb` 落地,smoke 16 passed,关键帧已内联进 `index.html`。** 视觉"够不够 Her"待浏览器肉眼确认。

### 1. 实现
- `frontend/src/engram.jsx`:新增 `THINKING_CSS`(4 组 `@keyframes` + `prefers-reduced-motion` 兜底)+ `ThinkingOrb` 组件;
  把 `sending` 指示器从 `<div>...<Mark/> Engram 正在想…</div>` 换成 `{sending && <ThinkingOrb />}`。
- 结构:`relative` 容器(28×28)→ 球后两道 `engOrbRipple` 涟漪环(第二道 `delay 1.4s` 错峰)→ 球体
  `radial-gradient(35% 30%, trace→jade→jadeInk)` + `engOrbBreathe`(呼吸+辉光)→ 球内 `conic-gradient` 浅 jade
  sheen + `engOrbSpin 5.5s` 缓转 → 右侧 `Engram 正在想…` 配 `engOrbText` 同步明灭。
- `@keyframes` 经组件内 `<style>{THINKING_CSS}</style>` 注入,仅 thinking 时挂载,**未改 `build.py`**。

### 2. 参数(便于后续微调)
| 元素 | 动画 | 时长/缓动 |
|---|---|---|
| 光球呼吸 | `scale 1⇄1.12` + jade `box-shadow` 浓淡 | `2.4s ease-in-out` |
| 内部 sheen 自转 | `conic-gradient` 旋转 | `5.5s linear` |
| 双涟漪 | `scale 0.55→2` + `opacity 0.5→0` | `2.8s ease-out`(第二道 delay 1.4s) |
| 文字明灭 | `opacity 0.5⇄1` | `2.4s ease-in-out`(与呼吸同步) |

呼吸 2.4s 与涟漪 2.8s 故意不整除 → 不锁步,更有机。
