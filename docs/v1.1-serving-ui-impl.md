## 改了什么(全部在 frontend/)
**`frontend/src/engram.jsx`(mock handler → fetch,只换数据源、不重构 JSX)**
- 新增 fetch helper(对齐 v1.0 契约表字段名):`apiConsolidateItem` / `apiDrop` / `apiRoute` / `apiPatch` /
  `apiEditModule` / `apiHealth`(原 `apiChat` / `apiConsolidate` / `apiMemories` 保留)。同源默认 `API=""`。
- `refresh()` 现在消费三层真值:`/memories` 的 `consolidated→weights`、`buffer→staged`(status/target 由
  `provenance.verdict` 派生:`supersede→"updates"`+`verdict_target_text`,否则 `"新增"`)、`rag→Reference`(此前 refs 是种子 mock)。
- 逐条按钮接真路由(按下→fetch→`refresh()`,后端真值不再被本地 state clobber):
  `burnOne→POST /consolidate/item`、`demoteOne→POST /memories/{id}/route {route:"rag"}`、
  `discardOne→POST /memories/{id}/drop`、`editPending`(本地保持输入流畅)+ blur 提交 `commitPending→PATCH /memories/{id}`。
- `editOn` 开关接 `POST /edit-module {on}`(乐观切换 + 失败回滚;无 adapter→409 回滚);启动时按 `/health.edit_on` 同步真态。
- `/chat` 扩展响应消费:`retrieved`→"retrieved from your documents · N 篇"徽章;`learned`→"记下了 · N 条新事实"。
- **诚实化**:Core memory 的删除按钮(Trash2)**移除**——codebook append-only、无单条 un-edit(v1.0 红线 / INV-S6),
  原"删掉就从模型拿掉"是假承诺;改文案为"由你在 Pending 区逐条决定写不写入"。Token 归因 / recalled 徽章 / 开场对话
  仍标注为 mock(本轮无后端 feed)。
- GPU 慢操作(approve-one ≈4.5s)期间 `consolidating` 置位,逐条按钮 disable 防重复提交。

**`frontend/build.py`(新增,可复现构建)**
- 此前"改 jsx → esbuild 重打包 → 重内联 index.html"是手工、未文档化的步骤。build.py 固化它:定位 esbuild
  (`/tmp/package/bin/esbuild`,缺则下 `@esbuild/linux-x64`)→ 备齐 pinned 依赖(react/react-dom/scheduler/lucide-react,
  缓存到 gitignored `frontend/.build/`,缺则从存活构建环境拷贝、再缺则 registry 下 tarball)→
  `esbuild --bundle --jsx=automatic --minify` → 内联 `vendor/tailwind.js`+`app.bundle.js` 生成自包含 `index.html`。
- 产物 `frontend/app.bundle.js`(180,608 B)、`frontend/index.html`(588,537 B)随源码一起提交。

**`tests/test_frontend_smoke.py`(重写,v0.6 mock-only → v1.1 wired)**
- 旧测断言"零 fetch / 含 babel+importmap"——v0.9 自包含内联 + 本轮接线后全已过时。新测断言:服务可达、
  9 个 api helper + `fetch(` + 9 条端点被引用、三层 `refresh()` 真消费、`index.html` 自包含(无外链 script / 无 babel)、
  **committed `index.html` 与源码同步**(内联里含 `/edit-module`·`/consolidate/item`·`/health`,防"改了 jsx 忘重建")、
  无客户端持久化(state 在后端内存,无 localStorage)。
